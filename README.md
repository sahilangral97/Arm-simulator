Instructions Implemented :-
Data Processing Instructions:
 ADD Rd, Rn, Rm
 SUB Rd, Rn, Rm
 AND Rd, Rn, Rm
 ORR Rd, Rn, Rm
 ADC Rd, Rn, Rm
 CMP Rn, Rm
 MOV Rd, Rm
 MNV Rd, Rm
Here operation are implemented with immediate also.

 LDR
 STR

 BLT
